# 知识点查询接口说明

## 新增接口

### 接口地址
```
GET /api/student/course/knowledge-points/{userId}/grade/{gradeId}/category/{categoryId}
```

### 功能说明
根据学生ID、年级和知识点分类查询知识点详情，包含学生的学习进度、难度等信息。

### 参数说明
- `userId`: 学生ID
- `gradeId`: 年级ID（例如：7=七年级，8=八年级，9=九年级）
- `categoryId`: 分类ID（例如：1=数与代数，2=几何，3=统计与概率，4=综合应用）

### 返回字段说明

#### 基本信息
- `pointName`: 知识点名称
- `description`: 知识点描述
- `content`: 知识点内容详情
- `difficultyLevel`: 难度等级（1=简单，2=中等，3=困难）
- `difficultyName`: 难度名称（简单/中等/困难）
- `difficultyDescription`: 难度描述

#### 学习进度
- `progressStatus`: 学习状态（1=未开始，2=学习中，3=已完成）
- `completionPercentage`: 完成百分比（0-100）
- `studyDuration`: 学习时长（分钟）
- `lastStudyTime`: 最后学习时间
- `startTime`: 开始学习时间
- `completeTime`: 完成学习时间
- `notes`: 学习笔记

#### 分类和年级
- `categoryName`: 分类名称
- `gradeName`: 年级名称
- `gradeLevel`: 年级等级

### 使用示例

**请求**
```bash
curl http://localhost:8080/api/student/course/knowledge-points/123/grade/7/category/1
```

**响应示例**
```json
{
  "code": 200,
  "message": "获取知识点详情成功",
  "data": [
    {
      "id": 101,
      "pointName": "正数和负数",
      "difficultyName": "简单",
      "difficultyLevel": 1,
      "progressStatus": 2,
      "completionPercentage": 75.00,
      "studyDuration": 60,
      "categoryName": "数与代数"
    }
  ]
}
```
