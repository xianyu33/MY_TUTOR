> 最后更新：2026-06-13

## 数据表总览

### 用户与角色管理（4张表）

| 表名 | 描述 | 关键字段 |
|------|------|----------|
| `user` | 学生账号 | user_account, email, email_verified, expire_time, role(S/P/T) |
| `parent` | 家长/老师账号 | user_account, type(0=家长, 1=老师) |
| `guardian_student_rel` | 监护人-学生绑定 | guardian_id, student_id, relation, start_at, end_at |
| `learning_statistics` | 每日学习统计 | user_id, statistics_date, total_study_time, average_score |

### 知识体系（4张表）

| 表名 | 描述 | 关键字段 |
|------|------|----------|
| `grade` | 年级（1-12） | grade_name, grade_level |
| `knowledge_category` | 知识点分类 | category_name, category_name_fr, category_code |
| `knowledge_point` | 知识点 | grade_id, category_id, point_name, difficulty_level, prerequisite_points |
| `knowledge_mastery` | 学生知识点掌握度 | user_id, knowledge_point_id, mastery_level, test_count |

### 题目与测试系统（6张表）

| 表名 | 描述 | 关键字段 |
|------|------|----------|
| `question` | 题库 | knowledge_point_id, question_type(1=选择, 2=填空, 3=解答), difficulty_level |
| `test` | 测试定义 | grade_id, knowledge_point_ids, total_questions, difficulty_level, test_type |
| `test_question` | 测试-题目关联 | test_id, question_id, sort_order, points |
| `student_test_record` | 学生测试记录 | student_id, test_id, earned_points, score_percentage, test_status |
| `student_test_answer` | 学生答题详情 | test_record_id, question_id, student_answer, is_correct, earned_points |
| `test_analysis_report` | AI 测试分析报告 | test_record_id, report_type, report_content, file_path |

### 学习进度（2张表）

| 表名 | 描述 | 关键字段 |
|------|------|----------|
| `learning_progress` | 知识点学习进度 | user_id, knowledge_point_id, progress_status, completion_percentage |
| `learning_content` | 学习内容记录 | user_id, knowledge_point_id, content_type(1=视频, 2=文档, 3=练习, 4=测试) |

### 支付系统（7张表）

| 表名 | 描述 | 关键字段 |
|------|------|----------|
| `payment_product` | Stripe 商品 | stripe_product_id, name, type, status |
| `payment_price` | Stripe 价格 | stripe_price_id, product_id, amount, currency, recurring_interval |
| `payment_customer` | Stripe 客户 | stripe_customer_id, user_id, email |
| `payment_order` | 支付订单 | order_no, customer_id, product_id, amount, status, stripe_session_id |
| `payment_subscription` | 订阅记录 | stripe_subscription_id, customer_id, status, current_period_start/end |
| `payment_refund` | 退款记录 | stripe_refund_id, order_id, amount, status, reason |
| `stripe_event` | Webhook 事件日志 | event_id, event_type, processed, processing_result |

## Schema 演进历史

### v1.0（2025-04）- 初始 Schema
- `user` 表（基础字段：账号、密码、角色）
- `chat_message` 和 `chat_message_detail` 表

### v1.1（2025-07）- 角色与通信
- 新增 `parent` 表（家长/老师账号）
- 新增 `guardian_student_rel`（绑定关系）
- `user` 表新增邮箱字段

### v2.0（2025-10）- 知识体系
- 新增 `grade`、`knowledge_category`、`knowledge_point` 表
- 新增 `question` 表（含难度等级）
- 新增 `test`、`test_question`、`student_test_record`、`student_test_answer`
- 新增 `learning_progress`、`learning_content`、`learning_statistics`
- 新增 `knowledge_mastery`（掌握度追踪）
- 新增 `test_analysis_report`（AI 分析报告）

### v2.1（2026-01）- 课程系统
- 新增课程相关表（Course, CourseContent）
- 新增学习计划表

### v3.0（2026-06）- 支付与安全
- 新增 `payment_product`、`payment_price`、`payment_customer`
- 新增 `payment_order`、`payment_subscription`、`payment_refund`
- 新增 `stripe_event`（Webhook 日志）
- `user` 表新增 `email_verified` 和 `expire_time` 字段

## 索引策略

| 表 | 索引 | 用途 |
|----|------|------|
| user | uk_user_account | 唯一账号查询 |
| user | idx_email | 邮箱校验查询 |
| user | idx_role | 角色过滤 |
| knowledge_point | idx_grade_category | 按年级+分类查知识点 |
| knowledge_point | idx_kp_difficulty | 按难度过滤 |
| student_test_record | idx_student_test | 学生测试历史 |
| student_test_record | idx_submit_time | 时间维度报表 |
| learning_progress | uk_user_knowledge | 每用户每知识点唯一进度 |
| payment_order | idx_customer_id | 客户订单查询 |
| payment_subscription | idx_stripe_subscription_id | Stripe Webhook 匹配 |
