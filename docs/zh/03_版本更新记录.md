## v3.0.0 - 支付与安全（2026-06-13）

**发布日期**：2026-06-13

### 新增
- **Stripe 支付体系集成**
  - 结账会话创建
  - 商品和价格管理
  - 订阅管理（创建、更新、取消）
  - Webhook 事件处理（15+ 事件类型）
  - 退款处理
  - 支付客户管理
  - 订单追踪和管理
  - Stripe 事件日志
  - 权益服务（含缓存）
- **邮箱校验链接直验**（点击即校验流程）
  - GET 端点用于邮件链接直接校验
  - POST 端点用于前端页面校验
  - 基于 Redis 的 Token 存储（24小时过期）
- **用户账号有效期**（注册默认一个月）
  - user 表新增 email_verified 字段
  - user 表新增 expire_time 字段
  - 登录校验检查邮箱验证和账号有效期

### 变更
- 邮箱校验链接直接指向后端 API
- 大文件上传优化（流式处理，无临时文件）
- Multipart 和 Tomcat 配置支持 500MB+ 文件上传

### 安全
- 移除意外暴露的 Stripe API Key
- 新增支付安全工具
- 受益人验证
