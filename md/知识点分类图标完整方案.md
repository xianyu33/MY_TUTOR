# 知识点分类图标完整方案

## ⚠️ 重要说明

Element UI 的图标在某些版本中可能不可用。本文档提供多个替代方案。

## 方案对比

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| Font Awesome | 图标丰富，社区支持好 | 需要引入外部CDN | ⭐⭐⭐⭐⭐ |
| SVG内联 | 完全可控，无需外部依赖 | 代码量大 | ⭐⭐⭐⭐ |
| Element Plus Icons | 官方维护，与Element UI风格一致 | 仅支持Vue 3 | ⭐⭐⭐⭐⭐ |
| IconFont | 自定义图标 | 需要自己准备图标文件 | ⭐⭐⭐ |

---

## 方案1: Font Awesome（最推荐）

### 引入Font Awesome

```html
<!-- 在 index.html 中添加 -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">
```

### 图标配置

#### 1. Number and Algebra（数与代数）

```html
<!-- 使用代码 -->
<i class="fas fa-calculator" style="color: #409EFF; font-size: 48px;"></i>

<!-- Vue组件 -->
<template>
  <i class="fas fa-calculator" :style="{ color: '#409EFF', fontSize: '48px' }"></i>
</template>
```

**备选图标**:
- `fa-calculator` (推荐) - 计算器
- `fa-math` - 数学公式
- `fa-plus-square` - 加号方块
- `fa-infinity` - 无穷符号

## 方案2: Element Plus Icons (适用于Vue 3)

### 安装

```bash
npm install @element-plus/icons-vue
```

### 使用

```vue
<template>
  <!-- 数与代数 -->
  <el-icon :size="48" :color="'#409EFF'">
    <Document />
  </el-icon>
  
  <!-- 几何 -->
  <el-icon :size="48" :color="'#67C23A'">
    <Grid />
  </el-icon>
  
  <!-- 统计与概率 -->
  <el-icon :size="48" :color="'#E6A23C'">
    <DataLine />
  </el-icon>
  
  <!-- 综合应用 -->
  <el-icon :size="48" :color="'#F56C6C'">
    <Operation />
  </el-icon>
</template>

<script>
import { Document, Grid, DataLine, Operation } from '@element-plus/icons-vue'

export default {
  components: {
    Document,
    Grid,
    DataLine,
    Operation
  }
}
</script>
```

## 总结

**推荐使用 Font Awesome**，因为：
1. ✅ 图标丰富且合适
2. ✅ 跨框架使用
3. ✅ CDN快速加载
4. ✅ 社区支持好
