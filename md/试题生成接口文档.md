# 试题生成接口文档

## 功能概述

根据年级和知识点自动生成50道测试题，支持文字和图文结合的选择题。

## API接口

### 生成测试题

**接口地址**: `POST /api/test/generate`

**功能说明**: 根据年级ID生成包含50道选择题的测试卷

**请求参数**:
```json
{
  "gradeId": 7,
  "difficultyLevel": 2,
  "questionCount": 50,
  "questionType": 1
}
```

**参数说明**:
- `gradeId`: 年级ID（必填）
- `difficultyLevel`: 难度等级（1=简单，2=中等，3=困难）
- `questionCount`: 题目数量，默认50
- `questionType`: 题目类型，1=选择题

**响应示例**:
```json
{
  "code": 200,
  "message": "测试题生成成功",
  "data": {
    "id": 1001,
    "testName": "七年级数学综合测试",
    "gradeId": 7,
    "totalQuestions": 50,
    "totalPoints": 50,
    "timeLimit": 60,
    "difficultyLevel": 2,
    "testType": 1,
    "status": 1,
    "createAt": "2023-12-21 10:00:00"
  }
}
```

## 试题类型

### 1. 纯文字选择题
题目内容仅包含文字描述，适合概念理解类题目。

**示例**:
```
计算：(-3)+5
A. -8  B. -2  C. 2  D. 8
正确答案：C
```

### 2. 图文结合选择题
题目包含图片和文字描述，适合几何图形题。

**示例**:
```
如图，直线a∥b，∠1=60°，则∠2的度数是：
[插入几何图形]
A. 60°  B. 120°  C. 30°  D. 90°
正确答案：A
```

## 试题分布建议

### 七年级（Grade 7）
- **数与代数**：20道
  - 正数和负数：3道
  - 有理数运算：8道
  - 科学记数法：2道
  - 近似数：2道
  - 综合应用：5道

- **几何**：15道
  - 基本图形：3道
  - 角的认识：4道
  - 平行线：5道
  - 综合应用：3道

- **统计与概率**：8道
  - 数据收集：2道
  - 数据整理：3道
  - 数据描述：3道

- **综合应用**：7道
  - 方程应用：3道
  - 不等式应用：2道
  - 综合题：2道

## 使用示例

```bash
curl -X POST http://localhost:8080/api/test/generate \
  -H "Content-Type: application/json" \
  -d '{
    "gradeId": 7,
    "difficultyLevel": 2,
    "questionCount": 50
  }'
```

## 注意事项

1. 题目支持多语言（中文/法语）
2. 选项数量建议4个（A、B、C、D）
3. 每道题分值为2分，总分100分
4. 建议考试时间：60分钟
5. 图文结合题需要提供图片URL
