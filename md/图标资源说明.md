# 知识点分类图标资源说明

## 重要说明 ⚠️

Element UI 的内置图标（如 `el-icon-s-data`）在某些版本中可能不可用。建议使用以下方案：

### 方案1: 使用 SVG 图标（推荐）
```vue
<template>
  <!-- 数与代数 -->
  <i class="custom-icon" style="color: #409EFF;">
    <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="currentColor">
      <path d="M19 3H5c-1.1 0-2 .9-2 2v14c0 1.1.9 2 2 2h14c1.1 0 2-.9 2-2V5c0-1.1-.9-2-2-2zm-5 14h-4v-2h4v2zm3-4H7v-2h10v2zm0-4H7V7h10v2z"/>
    </svg>
  </i>
</template>
```

### 方案2: 使用 Font Awesome
```html
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

<!-- 数与代数 -->
<i class="fas fa-calculator" style="color: #409EFF; font-size: 48px;"></i>

<!-- 几何 -->
<i class="fas fa-shapes" style="color: #67C23A; font-size: 48px;"></i>

<!-- 统计与概率 -->
<i class="fas fa-chart-bar" style="color: #E6A23C; font-size: 48px;"></i>

<!-- 综合应用 -->
<i class="fas fa-puzzle-piece" style="color: #F56C6C; font-size: 48px;"></i>
```

### 方案3: Element Plus Icons (Vue 3)
```javascript
import { Document, Grid, DataAnalysis, Link } from '@element-plus/icons-vue'

// 使用
<el-icon :size="48" color="#409EFF"><Document /></el-icon>
```

## Element UI图标列表

### 数据/统计类
- `el-icon-s-data` ← 推荐用于"数与代数"和"统计与概率"
- `el-icon-calculator` - 计算器
- `el-icon-s-marketing` - 营销

### 网格/图形类
- `el-icon-s-grid` ← 推荐用于"几何"
- `el-icon-view` - 查看
- `el-icon-rank` - 排名

### 应用/工具类
- `el-icon-s-opportunity` ← 推荐用于"综合应用"
- `el-icon-s-tools` - 工具
- `el-icon-s-help` - 帮助
