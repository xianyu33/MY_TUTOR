```html

<i class="el-icon-s-data" style="color: #409EFF; font-size: 48px;"></i>

<!-- 几何 -->
<i class="el-icon-s-grid" style="color: #67C23A; font-size: 48px;"></i>

<!-- 统计与概率 -->

<!-- 综合应用 -->
<i class="el-icon-s-opportunity" style="color: #F56C6C; font-size: 48px;"></i>
```

### Vue组件使用
```vue
<template>
  <div class="category-list">
    <div 
      v-for="category in categories" 
      :key="category.id"
      class="category-item"
      :style="{ borderTopColor: category.colorTheme.primary }"
    >
      <i :class="category.iconClass" class="category-icon"></i>
      <span>{{ category.name }}</span>
    </div>
  </div>
</template>

<style scoped>
.category-icon {
  font-size: 48px;
}
</style>
```

### 数据库更新
```sql

UPDATE knowledge_category SET
```

## 图标推荐

### 推荐使用的Element UI图标

1. **数与代数**
   - 首选：`el-icon-s-data`（数据图标）
   - 备选：`el-icon-calculator`（计算器）
   
2. **几何**
   - 首选：`el-icon-s-grid`（网格图标）
   - 备选：`el-icon-view`（查看）
   
3. **统计与概率**
   - 首选：`el-icon-s-data`（数据统计）
   - 备选：`el-icon-s-marketing`（营销统计）
   
4. **综合应用**
   - 首选：`el-icon-s-opportunity`（机会）
   - 备选：`el-icon-s-tools`（工具）

## 注意事项

1. 所有图标都是Element UI内置图标，无需额外引入
2. 颜色使用Element UI的默认主题色
3. 图标大小建议使用48px（2-3em）
4. 图片资源需要自己准备或使用默认图标
